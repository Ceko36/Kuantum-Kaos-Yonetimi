Bu dosya, Visual Studio'nun projeyi nasıl oluşturduğunu açıklar.

Bu projeyi oluşturmak için aşağıdaki adımlar kullanıldı:
- Proje dosyasını (`NodeConsoleApp1.esproj`) oluşturun.
- Hata ayıklamayı etkinleştirmek için `launch.json` dosyasını oluşturun.
- npm paketlerini yükleyin: `npm init && npm i --save-dev eslint`.
- `app.js` dosyası oluşturun.
- `package.json` giriş noktasını güncelleyin.
- Linting etkinleştirmek için `eslint.config.js` oluşturun.
- Projeyi çözüme ekleyin.
- Bu dosyayı yazın.
